  User Stories

  I want the user to be able to create a message that consists of 140 characters.
  I want the user to be able to save the message.
  I want the user to have the ability to edit the message.
  I want the user to have the ability to delete the message.
  I want the user to be able to comment on the message.
  I want the user to be able to share the message with a friend.
